# ☕ Cafe Website 

Welcome to my Cafe Website project! I built this full-stack web application to showcase my skills and learn modern web development. It features a fully responsive, modern user interface connected to a backend system to handle user data securely.

## 🚀 Overview
This project was developed from scratch as a personal portfolio piece. It demonstrates my ability to build clean, fast, and interactive user interfaces while managing server-side data storage.

## 🛠️ Technologies Used
I utilized a modern tech stack to ensure high performance, type safety, and great styling:
* **Next.js & React.js** – For building a fast, scalable frontend with component-based architecture.
* **TypeScript** – To write reliable, type-safe code and catch bugs early.
* **Tailwind CSS** – For crafting a modern, fully responsive UI that looks perfect on mobile, tablet, and desktop screens.
* **JavaScript & CSS** – For custom animations, functionality, and fine-tuning the user experience.
* **Backend Integration** – Connected to a database to efficiently store, manage, and retrieve user data (such as orders or messages).

## ✨ Key Features
* 📱 **100% Responsive Design:** Optimized for all screen sizes.
* 💾 **Data Persistence:** Fully operational backend that saves user inputs and data successfully.
* ⚡ **Fast Performance:** Server-side rendering and optimizations powered by Next.js.
* 🛠️ **Clean Code Architecture:** Well-organized folder structure following best development practices.
